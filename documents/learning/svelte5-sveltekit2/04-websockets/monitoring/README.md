## 学習の進め方

1. まず「WebSockets モニタリング概要」から始めて、モニタリングの基本概念を理解しましょう
2. 次に「WebSockets クライアント側モニタリング」で、クライアント側での実装方法を学びます

## モニタリング指標

- 接続数
- メッセージスループット
- レイテンシ
- エラー率
- 再接続回数
